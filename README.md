# FaunaShqip Django

FaunaShqip është një projekt web i ndërtuar me Django, i cili paraqet informacione rreth faunës në Shqipëri. Projekti përfshin faqe për listimin e kafshëve, detaje për secilën specie, blog informues dhe formë kontakti.

## Përshkrim i shkurtër

Ky projekt është realizuar si pjesë e lëndës Web Programming. Qëllimi i tij është krijimi i një faqeje web dinamike me Django, ku përdoruesi mund të shikojë informacione rreth kafshëve, të lexojë artikuj në blog dhe të përdorë formën e kontaktit.

## Funksionalitetet kryesore

- Faqe kryesore informuese
- Listë kafshësh
- Faqe detaji për kafshët
- Blog me artikuj informues
- Faqe kontakti
- Panel administrimi për menaxhimin e të dhënave

## Struktura kryesore e projektit

- `animals/` – aplikacioni kryesor i projektit
- `faunashqip/` – konfigurimi kryesor i Django project
- `templates/` – faqet HTML të projektit
- `static/` – file-t CSS dhe elementët statikë
- `media/` – imazhet e përdorura në projekt
- `manage.py` – file kryesor për komandat e Django-s
